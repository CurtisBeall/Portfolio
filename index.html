Details

This mock website was designed for the restaurant Pacific Tavern by George Zhang. It was built with HTML, styled with CSS and heavily uses jQuery to perform functions throughout the site.


This site was built to be mobile-first and fully responsive across any desktop or mobile device. On mobile devices and smaller screens users use a hidden side-bar menu by clicking on an arrow at the top of the screen. This site is designed so you never leave the page. Instead it uses smooth section scrolling to navigate up and down the page.


Also featured is a working reservation modal built from scratch. It was built to limit users from setting a reservation on the wrong day with too many or few guests. It also uses filed validation as well to ensure are fields are correct. Also built from scratch is an image carousal in the Gallery section that showcases the restaurant. For ease of updating the Menu and Drink sections they are auto populated from outside lists.

Details

This mock website was designed for the shoe company Kickasso by Filip Slováček. Built with HTML, this site was styled with CSS and Bootstrap. The majority of the functions on this site were coded from scratch, the exception being on the product pages, where a modified Bootstrap image carousal is used to showcase the products.


Designed for simplicity, this site uses a hamburger side-bar menu whether users are on a desktop or mobile device. This made it easier to build this site to be mobile-first and fully responsive.


Also featured is a cart function built from scratch. Products can be added, removed, and have their counts changed in real time. The cost is also updated in real time on the cart page while the number of different items in your cart can be seen throughout the site I upper right corner in real time. On the Shop page there is a filtering system that was built from scratch. A list of the filtered items are used for next and previous products when navigating on the product page.

Details

This website is designed for a fictional agency called Splash. It was built without the use of any third party frameworks or libraries. This was an intentional decision as I wanted to see what it would be like to not use such resources. A hamburger menu, created by Nick Salloum, was used in developing this site.


Designed to be fully responsive this website utilizes the hamburger drop-down menu for mobile devices and smaller screens. The Work page use an algorithm in the CSS calculation to easily change the rows.


Also featured is parallax scrolling on the Home page and interactive animation on the About page. The animation, linked to the scroll position, moves text information on the page to hide and reveal them from behind their respective images.
